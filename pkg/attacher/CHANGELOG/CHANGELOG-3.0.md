# Release notes for v3.0.0

[Documentation](https://kubernetes-csi.github.io/docs/)

# Changelog since v2.2.0

## Urgent Upgrade Notes 

### (No, really, you MUST read this before you upgrade)

- Update volumeAttachment to v1
  
  RBAC policy was updated to allow the external-attacher to patch VolumeAttachment.Status ([#200](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/200), [@cwdsuzhou](https://github.com/cwdsuzhou))
 - Use GA version of CSINode object. The external-attacher now requires Kubernetes 1.17. ([#193](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/193), [@bertinatto](https://github.com/bertinatto))
 
## Changes by Kind

### Feature

- Added support for migration of Kubernetes in-tree VMware volumes to CSI. ([#236](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/236), [@divyenpatel](https://github.com/divyenpatel))

### Bug or Regression

- Fixes an issue in volume attachment reconciler when the CSI driver supports LIST_VOLUMES_PUBLISHED_NODES but does not implement CSI migration. ([#244](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/244), [@yuga711](https://github.com/yuga711))
- Use dedicated Kubernetes client for leader election that does not get throttled when the external-attacher is under heavy load. ([#242](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/242), [@jsafrane](https://github.com/jsafrane))

### Other (Cleanup or Flake)

- Removed support of go dep. ([#239](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/239), [@jsafrane](https://github.com/jsafrane))

### Uncategorized

- Build with Go 1.15 ([#246](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/246), [@pohly](https://github.com/pohly))
- Publishing of images on k8s.gcr.io ([#231](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/231), [@pohly](https://github.com/pohly))
- Updated client-go to v0.18 ([#221](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/221), [@humblec](https://github.com/humblec))
