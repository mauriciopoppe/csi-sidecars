# Release notes for v3.3.0

[Documentation](https://kubernetes-csi.github.io)
# Changelog since v3.2.0

## Changes by Kind

### Feature

- Add command line arguments to configure leader election options ([#313](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/313), [@RaunakShah](https://github.com/RaunakShah))
- Adds mappings for PV access modes to new CSI access modes: `SINGLE_NODE_SINGLE_WRITER` and `SINGLE_NODE_MULTI_WRITER`. ([#308](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/308), [@chrishenzie](https://github.com/chrishenzie))
- Updates Kubernetes dependencies to v1.22.0 ([#321](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/321), [@chrishenzie](https://github.com/chrishenzie)) [SIG Storage]

### Bug or Regression

- Fix a bug that the controller can panic crash when it receives DeletedFinalStateUnknown deletion event. ([#304](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/304), [@Jiawei0227](https://github.com/Jiawei0227))

### Other (Cleanup or Flake)

- Updates container-storage-interface dependency to v1.5.0 ([#312](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/312), [@chrishenzie](https://github.com/chrishenzie))
- Reuse the same gRPC CSI client for all CSI driver calls ([#318](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/318), [@yeya24](https://github.com/yeya24))
