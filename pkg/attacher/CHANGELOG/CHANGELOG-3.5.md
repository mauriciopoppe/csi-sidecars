# Release notes for v3.5.0

[Documentation](https://kubernetes-csi.github.io)
# Changelog since v3.4.0

## Changes by Kind

### Bug or Regression

- There is a new --default-fstype flag available in this release that defaults to ext4 and can be used to set any value a driver may need. In the next major release ext4 as the default fsType will be deprecated and replaced by "" ([#342](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/342), [@RomanBednar](https://github.com/RomanBednar))
- Upgrade csi-translation-lib for some bug fixes with azuredisk and azurefile ([#352](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/352), [@andyzhangx](https://github.com/andyzhangx))

### Uncategorized

- Kube client dependencies are updated to v1.24.0 ([#353](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/353), [@humblec](https://github.com/humblec))
