# Changelog since v3.0.0

## Changes by Kind

### Other (Cleanup or Flake)

- `klog/v2 is` used for logging
- `process_start_time` metric is now reported ([#277](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/277), [@jsafrane](https://github.com/jsafrane))
- The `metrics-address` flag is deprecated and replaced by `http-endpoint`, which enables handlers from both metrics manager and leader election health check. ([#279](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/279), [@verult](https://github.com/verult))
