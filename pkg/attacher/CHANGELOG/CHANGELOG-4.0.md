# Changelog since v3.5.0

## Urgent Upgrade Notes

### (No, really, you MUST read this before you upgrade)

- Change `--default-fstype` from `ext4` to empty string. CSI drivers that depended on ext4 as the default need to change their deployments to explicitly set `--default-fstype=ext4`. ([#358](https://github.com/kubernetes-csi/csi-sidecars/pkg/attacher/pull/358), [@Kartik494](https://github.com/Kartik494))
